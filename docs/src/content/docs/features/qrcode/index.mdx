# QR Code

Mercury provides a `<QRCode>` component to generate QR codes.
This allows you to easily embed QR codes within your slides.

## Basic Usage

You can generate a QR code by passing the desired value to the `value` property.

```mdx
<QRCode value="https://mercury.r4ai.dev" />
```

This component is imported by default, so you don't need to import it explicitly.

However, you can also import it manually as follows:

```mdx
import { QRCode } from "@mercurymd/react"

<QRCode value="https://mercury.r4ai.dev" />
```

## Customizing Appearance

### Size

You can control the rendered width and height in pixels using the `size` property.

```mdx
<QRCode value="https://mercury.r4ai.dev" size={200} />
```

### Styling with Tailwind CSS

The background and foreground colors can be changed using Tailwind CSS utility classes.

```mdx
<QRCode className="border text-blue-600 bg-blue-50" value="https://mercury.r4ai.dev" />
```

## Advanced Options

For more details, please refer to the following page:

https://github.com/soldair/node-qrcode/tree/v1.5.4?tab=readme-ov-file#options-9

### version

- Type: `Number`

QR Code version. If not specified, the most suitable value will be calculated.

### errorCorrectionLevel

- Type: `String`
- Default: `M`

Error correction level.
Possible values are `low`, `medium`, `quartile`, `high` or `L`, `M`, `Q`, `H`.

### maskPattern

- Type: `Number`

Mask pattern used to mask the symbol.
Possible values are `0`, `1`, `2`, `3`, `4`, `5`, `6`, `7`.
If not specified, the most suitable value will be calculated.

### toSJISFunc

- Type: `Function`

A helper function used internally to convert a kanji to its Shift JIS value.
Provide this function if you need support for Kanji mode.
