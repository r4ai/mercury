# Mathematics

Mercury は $\LaTeX$ による数式記述をサポートしています。

## インライン数式

インライン数式は `$...$` で囲むことで記述できます。

```mdx
これはインライン数式の例です: $E=mc^2$。
```

## ブロック数式

ブロック数式は `$$...$$` で囲むことで記述できます。

```mdx
$$
\int_a^b f(x) \,dx = F(b) - F(a)
$$
```

## 対応している $\LaTeX$ コマンドについて

Mercury では数式のレンダリングに $\KaTeX$ を使用しています。
対応している $\LaTeX$ コマンドについては次ののページを参照してください。

https://katex.org/docs/supported.html

## オプション

数式のレンダリングには remark-math と rehype-katex を使用しています。
それぞれのオプションの指定方法は以下の通りです。

### remark-math

remark-math のオプションは `remarkMath` プロパティで指定します。

```ts title="vite.config.ts" {11-13}
import mercury from "@mercurymd/vite-plugin";
import tailwindcss from "@tailwindcss/vite";
import react from "@vitejs/plugin-react";
import { defineConfig } from "vite";

// https://vitejs.dev/config/
export default defineConfig({
  plugins: [
    mercury({
      mdx: {
        remarkMath: {
          // write your options here...
        },
      },
    }),
    react({ include: /\.(mdx|js|jsx|ts|tsx)$/ }),
    tailwindcss(),
  ],
});
```

利用可能なオプションについては次のページを参照してください。

https://github.com/remarkjs/remark-math

### rehype-katex

rehype-katex のオプションは `rehypeKatex` プロパティで指定します。

```ts title="vite.config.ts" {11-13}
import mercury from "@mercurymd/vite-plugin";
import tailwindcss from "@tailwindcss/vite";
import react from "@vitejs/plugin-react";
import { defineConfig } from "vite";

// https://vitejs.dev/config/
export default defineConfig({
  plugins: [
    mercury({
      mdx: {
        rehypeKatex: {
          // write your options here...
        }
      },
    }),
    react({ include: /\.(mdx|js|jsx|ts|tsx)$/ }),
    tailwindcss(),
  ],
});
```

利用可能なオプションについては次のページを参照してください。

https://github.com/remarkjs/remark-math/tree/main/packages/rehype-katex

https://katex.org/docs/options.html
