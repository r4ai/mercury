# Features

- MDXでスライドを作成できる
- MDXからスライドを表示するHTML（ウェブアプリ）を生成する

<br />

- Reactコンポーネントを使ってスライドをカスタマイズできる
- TailwindCSSを使ってスライドをスタイリングできる
- 数式を表示できる
- コードブロックを表示できる

# Example

```tsx title=main.tsx
import { Router } from "wouter";

// import時に、MDXからJSX(スライド用コンポーネント)へ変換される
import Presentation from "./Presentation.mdx";

ReactDOM.createRoot(document.getElementById("root")!).render(
  <React.StrictMode>
    <Router>
      <ThemeProvider attribute="data-color-scheme">
        <Presentation />
      </ThemeProvider>
    </Router>
  </React.StrictMode>,
);
```
